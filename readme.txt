Перед началом работы с программой требуется:
1. Установить зависимости requirements.txt
    1.1. Перейти в командной строке в корневую папку проекта
    2.2. pip install -r requirements.txt
2. Запустить файл /MLAudioAI/app.py (С помощью любой среды разработки/терминала)
~~~~
Чтобы добавить нужные аудиофайлы, добавьте их в каталог /MLAudioAI/audioVal
Датасеты для обучения модели нужно добавлять в /MLAudioAI/data (предваритаельно создав его)
Модели нейросети находятся в /MLAudioAI/newAudioModels
Ссылка на датасет: https://huggingface.co/datasets/DynamicSuperbPrivate/SpoofDetection_Asvspoof2017
Перед использованием функций импорта датасета требуется указать актуальные пути к файлам датасета : /MLAudioAI/files.py

Перед ипортом аудио из датасетов требуется удалить актуальную папку с уже использовавшимися аудиофайлами (если есть): /audioVal

Реализованные функции:
	- Создание модели данных на основе реальных датасетов
	- Обучение модели нейронной сети с последующей компиляцией и сохранением
	- Загрузка аудиофайлов в формате .wav/.mp3
	- преобразование файла в спектрограмму
	- анализ спектрограммы нейронной сетью 
	- вывод предсказания является ли аудиозапись дипфейком или нет
Особенность проекта в следующем:
	- Наглядный вывод спектрограмм для аудио;
	- Поддержка множества форматов аудио;
	- >90% верных предсказаний
Основной стек технологий:
	- Python 3.12
	- Git
	- HTML, CSS
Среда запуска:
	- Windows 8.1 и новее
	- требуется установленный Python 3.12
