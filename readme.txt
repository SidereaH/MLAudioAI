Чтобы добавить нужные аудиофайлы, добавьте их в каталог /PyCharmMain/audioVal
Датасеты для обучения модели находятся в  /PyCharmMain/data
Модели нейросети находятся в /PyCharmMain/


Реализованные функции:
	- Создание модели данных на основе реальных датасетов
	- Обучение модели нейронной сети с последующей компиляцией и сохранением
	- Загрузка аудиофайлов в формате .wav/.mp3
	- преобразование файла в спектрограмму
	- анализ спектрограммы нейронной сетью 
	- вывод предсказания является ли аудиозапись дипфейком или нет
Особенность проекта в следующем:
	- Наглядный вывод спектрограмм для аудио;
	- Поддержка множества форматов аудио;
	- Киллерфича-3;
Основной стек технологий:
	- Python 3.12
	- Git
Среда запуска:
	- Windows 8.1 и новее
	- требуется установленный Python 3.12
